Flexbox Froggy Pro

CSS FLEXBOX
PRO

Now that you've learned the fundamentals of CSS flexbox, it's time to become a master. Face endless rounds of practice, with levels that ramp up in difficulty and pinpoint your weaknesses. Game reports also help you reflect on your progress. Flexbox Froggy Pro introduces 8 ponds, each with a mix of 20 levels. Replay a pond and face completely new levels each time. Challenge yourself to beat them all!

Learning objectives

Practice and improve recall of CSS flexbox properties and values.

Prerequisites

For those experienced of CSS flexbox. You should complete Flexbox Froggy for an introduction before playing this game.
